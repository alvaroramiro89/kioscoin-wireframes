REQUERIMIENTOS KIOSCOIN V2 - PROFILE SCREEN
============================================

INSPIRACIÓN: App Revolut (navegación móvil moderna)
OBJETIVO: Interfaz simple, limpia, sin scroll, todo visible en una pantalla

ESTRUCTURA GENERAL
==================

1. BARRA DE NAVEGACIÓN INFERIOR (como Revolut)
----------------------------------------------
Posición: Fixed bottom
3 botones principales:
- INICIO (logo Kioscoin + ícono casa)
- MIS ÓRDENES (ícono órdenes/transacciones)
- MI CUENTA/PROFILE (ícono usuario)

Estilo:
- Fondo blanco/gris claro
- Íconos minimalistas
- Botón activo resaltado
- Sombra sutil hacia arriba

2. SECCIÓN WALLET DEL USUARIO (sin conectar wallet)
--------------------------------------------------
- Mostrar wallet que YA pertenece al usuario
- Dirección de wallet visible (truncada)
- 2 BALANCES principales:
  * USDK (balance + valor en fiat)
  * USDC (balance + valor en fiat)
- Botón de refresh/actualizar balances

3. SECCIÓN CUENTAS BANCARIAS/MONEDEROS
-------------------------------------
Título: "Cuentas y Monederos"
Lista de métodos de pago:
- Revolut
- ING
- N26
- WISE
- Botón "+ Agregar cuenta"

Cada cuenta muestra:
- Logo del banco
- Nombre de la cuenta
- Estado (verificada/pendiente)

4. SECCIÓN COMPRA/VENTA RÁPIDA
-----------------------------
Cards de acciones rápidas:
- "Comprar USDC" (botón verde)
- "Vender USDC" (botón rojo)
- "Comprar USDK" (botón verde)
- "Vender USDK" (botón rojo)

Layout: 2x2 grid de cards

ESPECIFICACIONES TÉCNICAS
=========================

TECNOLOGÍA:
- HTML + CSS puro (para mockup rápido)
- Responsive design
- Sin scroll vertical (height: 100vh)

PALETA DE COLORES (BRANDKIT):
- Lila Kioscoin: #8B5CF6 (primary)
- Lavanda: #A78BFA (secondary)
- Azul-violeta: #6366F1 (accent)
- Blanco: #FFFFFF
- Grises: #F9FAFB a #111827
- Success: #10B981
- Error: #EF4444

LAYOUT:
- Sin header tradicional
- Navegación bottom fixed
- Contenido principal scrolleable solo si necesario
- Máximo uso del viewport

FASE 1 - IMPLEMENTACIÓN INICIAL
===============================

EMPEZAR SOLO CON:
1. Barra de navegación inferior (3 botones)
2. Estructura básica HTML
3. CSS para la navegación

PRÓXIMAS FASES:
- Fase 2: Sección wallet + balances
- Fase 3: Cuentas bancarias
- Fase 4: Cards compra/venta
- Fase 5: Interactividad y estados

REFERENCIAS VISUALES
===================

INSPIRACIÓN REVOLUT:
- Navegación inferior limpia
- Cards con bordes redondeados
- Espaciado generoso
- Tipografía clara y legible
- Íconos minimalistas

DIFERENCIAS CON KIOSCOIN V1:
- NO más "Conectar wallet"
- NO header superior
- NO scroll innecesario
- Navegación móvil nativa
- Enfoque en crypto específico (USDC/USDK)

NOTAS IMPORTANTES
================

1. SIMPLICIDAD: Pocas cosas por vez, bien hechas
2. MOBILE FIRST: Diseño pensado para móvil
3. SIN SCROLL: Todo visible en una pantalla
4. USABILIDAD: Acciones principales fácilmente accesibles
5. PROGRESIVO: Construir iterativamente
